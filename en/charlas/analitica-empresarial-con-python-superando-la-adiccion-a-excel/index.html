WORKSHOP Analítica Empresarial con Python: superando la adicción a Excel TOOLS
   Language:   Spanish    Duration:    2 hours    Level:   Intermediate

La Analítica de Negocios se ha asociado tradicionalmente al uso de Excel para generar reportes estratégicos de gestión, bajo la premisa implícita que las destrezas de programación son difíciles de desarrollar o que hay herramientas que permiten evitarlas. En la actualidad, carecer de estas destrezas imponen de restricciones excesivas, en particular cuando requerimos automatizar tareas o realizar operaciones complejas en entornos abiertos y distribuidos. En este taller se va a explorar el uso de Python y el stack de análisis de datos como sustituto de las operaciones que se realizan usualmente desde las hojas de cálculo. Se revisarán los principales paquetes utilizados para el análisis exploratorio de datos hasta los métodos fundamentales de análisis multivariante y aprendizaje autómatico, incorporando las principales herramientas de visualización y de generación de reportes. Debido a las limitaciones de tiempo se hará énfasis en algunas características avanzadas de herramientas como Pandas, Simpy, OR-Tools, Bokeh, Plotly, Statmodels y sk-learn, entre otros en base a JupyterLab, para obtener resultados atractivos en el mundo profesional.
